# Overview

This guide prepares a fresh Microsoft 365 tenant for the **Data Security Hack** lab. By the end of step 03 you will have:

- A Visual Studio Enterprise subscription assigned in Partner Center
- A provisioned **Microsoft 365 E5 Instant Sandbox** with admin credentials and 16 sample users
- A dedicated Microsoft Edge profile signed in as the sandbox admin
- The `HR` Microsoft 365 group created and the `U.S. Sales` group adjusted
- Tenant-wide audit logging and sensitivity-label support enabled (Unified Audit Log, SharePoint AIP integration, Office co-authoring, container labels for Groups/Teams/Sites)

## Why an Instant Sandbox?

The Microsoft 365 Developer Program **Instant Sandbox** is a fully pre-configured E5 environment that provisions in minutes. It includes:

- 16 fictitious sample users
- Microsoft Teams data packs
- Microsoft Graph and Office Add-ins data
- Pre-configured custom apps
- Add-on commerce — purchase additional services (including Microsoft Copilot) directly from the sandbox

> **Important:** You are not charged for the free E5 developer sandbox. Billing details are required only for account verification and to enable future add-on purchases.
